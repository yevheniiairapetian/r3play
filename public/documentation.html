R3:play app! Documentation

Features

  1. Return a list of ALL movies to the user
  2. Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user
  3. Return data about a genre (description) by name/title (e.g., “Thriller”)
  4. Return data about a director (bio, birth year, death year) by name
  5. Allow new users to register
  6. Allow users to update their user info (username, password, email, date of birth)
  7. Allow users to add a movie to their list of favorites
  8. Allow users to remove a movie from their list of favorites
  9. Allow existing users to deregister

To return a movie genre name and description:

Use the resource URl: https://github.com/yevheniiairapetian/r3play/movies/:genres/:GenreName

Response formats JSON
Request Metod Type GET
Query parameters Genre of the movie in the format https://github.com/yevheniiairapetian/r3play/movies/genres/:GenreName where :GenreName is replaced with the desired genre
Example request https://github.com/yevheniiairapetian/r3play/movies/genres/Comedy
Example response { "Name": "Comedy",
"Description": "A comedy film is a category of film which emphasizes on humor. These films are designed to make the audience laugh in amusement" }

To return a movie director's name and description:

Use the resource URl: https://github.com/yevheniiairapetian/r3play/movies/directors/:DirectorName

Response formats JSON
Request Metod Type GET
Query parameters Genre of the movie in the format movies/:directors/:DirectorName where :DirectorName is replaced with the desired movie director's name
Example request https://github.com/yevheniiairapetian/r3play/movies/Directors/The%20Wachowskis
Example response { "Name": "The Wachowskis",
"Bio": "Lilly Wachowski and Lana Wachowski are American film and television directors, writers, and producers." }

To create a new user:

Use the resource URl: https://github.com/yevheniiairapetian/r3play/users

Response formats JSON
Request Metod Type POST
Query parameters Enter the user name, password, email (all required) and date of birthday (optional) as JSON object
Example request https://github.com/yevheniiairapetian/r3play/users
JSON request:
{ "Username": "Johny",
"Password": "12345",
"Email": "johny321@gmail.com" }
Example response { "Username": "Johny",
"Password": "12345",
"Email": "johny321@gmail.com",
"FavoriteMovies": [],
"_id": "64c2c3acc5aaaf1c491bdb6e",
"__v": 0 }

To update a user name:

Use the resource URl: https://github.com/yevheniiairapetian/r3play/users/:id

Response formats JSON
Request Metod Type PUT
Query parameters Update the user name, password, email and/or date of birthday as JSON object
Example request https://github.com/yevheniiairapetian/r3play/users/Johny
JSON request:
{ "Username": "Johny1",
"Password": "1232245",
"Email": "johny321@gmail.com" }
Example response { "_id": "64c2c3acc5aaaf1c491bdb6e",
"Username": "Johny1",
"Password": "1232245",
"Email": "johny321@gmail.com",
"FavoriteMovies": [],
"__v": 0 }

To delete a user:

Use the resource URl: https://github.com/yevheniiairapetian/r3play/users/:id

Response formats Text message
Request Metod Type DELETE
Query parameters Enter the user name in the query
Example request https://github.com/yevheniiairapetian/r3play/users/Johny1
Example response Johny1 was deleted.

Use the resource URl: https://github.com/yevheniiairapetian/r3play/users/:id/:movies/:MovieID

Response formats Text message
