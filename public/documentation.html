R3play API Documentation

R3play API Github repository

Please refer for more information to the R3play API Github repository

Features

  1. Return a list of ALL movies, tv series and anime to the user
    Return data for movies:

    title {string}
    description {string}
    genre (name {string}, description {string})
    director (name {string}, bio {string}, birth year {string}, death year {string})
    image URL {string}
    featured (true, false) {string}
    actors {array}
    Rotten Tomatoes Audience rating {string}
    IMDb rating {string}
    release date {string}
    duration {string}
    trailer url {string}



  2. for tv series and anime:

    title {string}
    description {string}
    genre (name {string}, description {string})
    director (name {string}, bio {string}, birth year {string}, death year {string})
    image URL {string}
    featured (true, false) {string}
    actors {array}
    Rotten Tomatoes Audience rating {string}
    IMDb rating {string}
    release date {string}
    episodes {array}
    duration {string}
    trailer url {string}

  3. Return data about a genre (name, description) by name/title (e.g., “Thriller”)
  4. Return data about a director (bio, birth year, death year) by name
  5. Allow new users to register
  6. Allow users to update their user info (username, password, email, date of birth)
  7. Allow users to add a movie, tv series, anime to their list of favorites
  8. Allow users to remove a movie, tv series, anime from their list of favorites
  9. Allow existing users to deregister

User Section

To create a new user:

Use the resource URl: https://r3play-934f9ea5664d.herokuapp.com/users

Response formats JSON
Request Metod Type POST
Query parameters Enter the user name, password, email (all required) and date of birthday (optional) as JSON object
Example request https://r3play-934f9ea5664d.herokuapp.com/users
JSON request:
{ "Username": "Johny",
"Password": "12345",
"Email": "johny321@gmail.com" }
Example response { "Username": "Johny",
"Password": "12345",
"Email": "johny321@gmail.com",
"FavoriteMovies": [],
"_id": "64c2c3acc5aaaf1c491bdb6e",
"__v": 0 }

To login

Use the resource URl: https://r3play-934f9ea5664d.herokuapp.com/login

Response formats JSON
Request Metod Type POST
Query parameters Enter the user name and password in the login form along with the personal token generated for the user
Example request https://r3play-934f9ea5664d.herokuapp.com/login
JSON request:
{ "Username": "Johny",
"Password": "12345",
}
Example response { "Username": "Johny",
"Password": "12345",
"Email": "johny321@gmail.com",
"FavoriteMovies": [],
"_id": "64c2c3acc5aaaf1c491bdb6e",
"__v": 0 }

To update a user name:

Use the resource URl: https://r3play-934f9ea5664d.herokuapp.com/users/:id

Response formats JSON
Request Metod Type PUT
Query parameters Update the user name, password, email and/or date of birthday as JSON object
Example request https://r3play-934f9ea5664d.herokuapp.com/users/Johny
JSON request:
{ "Username": "Johny1",
"Password": "1232245",
"Email": "johny321@gmail.com" }
Example response { "_id": "64c2c3acc5aaaf1c491bdb6e",
"Username": "Johny1",
"Password": "1232245",
"Email": "johny321@gmail.com",
"FavoriteMovies": [],
"__v": 0 }

To delete a user:

Use the resource URl: https://r3play-934f9ea5664d.herokuapp.com/users/:id

Response formats Text message
Request Metod Type DELETE
Query parameters Enter the user name in the query
Example request https://r3play-934f9ea5664d.herokuapp.com/users/Johny1
Example response Johny1 was deleted.
