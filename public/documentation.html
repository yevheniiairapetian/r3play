R3:play app! Documentation

Features

  1. Return a list of ALL movies to the user
  2. Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user
  3. Return data about a genre (description) by name/title (e.g., “Thriller”)
  4. Return data about a director (bio, birth year, death year) by name
  5. Allow new users to register
  6. Allow users to update their user info (username, password, email, date of birth)
  7. Allow users to add a movie to their list of favorites
  8. Allow users to remove a movie from their list of favorites
  9. Allow existing users to deregister

To return a list of ALL movies:

Use the resource URl: https://github.com/yevheniiairapetian/r3play/movies

Response formats JSON
Request Metod Type GET
Request Metod Type GET
Query parameters None
Example request https://github.com/yevheniiairapetian/r3play/movies
Example response { name: "Big", releaseDate: "June 3, 1988", actors: ["Tom Hanks", "Elizabeth Perkins", "Robert Loggia", "John Heard"], director: "Penny Marshall", rottenTomatoesRate: 98 }, { name: "Inception", releaseDate: "July 8, 2010", actors: [ "Leonardo DiCaprio", "Ken Watanabe", "Joseph Gordon-Levitt", "Marion Cotillard", "Elliot Page", "Tom Hardy", "Cillian Murphy", "Tom Berenger", "Michael Caine" ], director: "Christopher Nolan", rottenTomatoesRate: 87 }

To return a specific movie by name:

Use the resource URl: https://github.com/yevheniiairapetian/r3play/movies/:name

Response formats JSON
Request Metod Type GET
Query parameters Name of the movie in the format movies/:name where :name is replaced with the desired movie
Example request https://github.com/yevheniiairapetian/r3play/movies/Inception
Example response { name: "Inception", releaseDate: "July 8, 2010", actors: [ "Leonardo DiCaprio", "Ken Watanabe", "Joseph Gordon-Levitt", "Marion Cotillard", "Elliot Page", "Tom Hardy", "Cillian Murphy", "Tom Berenger", "Michael Caine" ], director: "Christopher Nolan", rottenTomatoesRate: 87 }

To return a movie genre description:

Use the resource URl: https://github.com/yevheniiairapetian/r3play/movies/:genres

Response formats JSON
Request Metod Type GET
Query parameters Genre of the movie in the format movies/:name/:genres where :genres is replaced with the desired genre
Example request https://github.com/yevheniiairapetian/r3play/movies/Inception/Thrillers
Example response Thriller

To return a movie director's description:

Use the resource URl: https://github.com/yevheniiairapetian/r3play/movies/:name/:directors

Response formats JSON
Request Metod Type GET
Query parameters Genre of the movie in the format movies/:name/:directors where :directors is replaced with the desired movie director(s) name(s)
Example request https://github.com/yevheniiairapetian/r3play/movies/inception/directors
Example response { "Christopher Nolan"}

To create a new user:

Use the resource URl: https://github.com/yevheniiairapetian/r3play/users

Response formats JSON
Request Metod Type POST
Query parameters Enter the user name as JSON object
Example request https://github.com/yevheniiairapetian/r3play/users
Example response User John was successfully created

To update a user name:

Use the resource URl: https://github.com/yevheniiairapetian/r3play/users

Response formats Text message
Request Metod Type PUT
Query parameters Enter the new user name as query parameter
Example request https://github.com/yevheniiairapetian/r3play/Jake
Example response User name succesfully changed to Jake

To delete a user:

Use the resource URl: https://github.com/yevheniiairapetian/r3play/users

Response formats Text message
Request Metod Type DELETE

Example request https://github.com/yevheniiairapetian/r3play/John
